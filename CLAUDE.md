# MARIE — Portfolio site

Статический сайт-портфолио фотографа. Без сборки, без зависимостей — открывается прямо из файлов (либо через локальный сервер IDE).

## Структура

```
index.html          — главная (галерея)
about.html          — обо мне
pricing.html        — цены
client-guide.html   — гайд клиенту
script.js           — галерея + лайтбокс (грузит data/photos.json)
pricing.js          — цены (грузит data/pricing.json)
style.css           — основные стили
pricing.css         — стили страницы цен

data/
  photos.json       — список фото галереи (массив объектов!)
  pricing.json      — данные для pricing.html
  about.json        — данные для about.html

photos/             — все фото галереи
video/              — фоновые/демо видео (Vimeo основное)
icons/              — иконки (cursor, cross, favicon)
font/               — шрифты
```

## Добавление новых фото — ВАЖНО

Исходники присылаются в `~/Documents/portfolio/` (вне репо). Порядок действий:

1. **Скопировать** файл в `photos/`:
   ```bash
   cp ~/Documents/portfolio/<имя>.jpg photos/
   ```

2. **Сжать** перед коммитом. Целевой размер — как у соседей: ширина ~1600px, качество 80, итог ~0.5–1.5 МБ. Сырые исходники с камеры (10–15 МБ, 4500×5700) на сайт класть НЕЛЬЗЯ:
   ```bash
   sips --resampleWidth 1600 -s formatOptions 80 photos/<имя>.jpg --out photos/<имя>.jpg
   ```

3. **Добавить запись в `data/photos.json`** — это массив **объектов**, не строк. Минимальная запись:
   ```json
   { "src": "<имя>.jpg" }
   ```
   Опциональные поля:
   - `"big": true` — крупная карточка в галерее
   - `"main": true` — выносится в верхний main-row (видео/первое фото)
   - `"type": "video"` + `"videoId": "<vimeoId>"` — видео из Vimeo
   - Для локального видео: `"src": "video/<file>.mp4"` + `"type": "video"`

   Порядок в массиве = порядок на сайте. Вставлять рядом с тематически близкими (например, новый `DSC086xx` — между `DSC08172` и `DSC08825`).

4. **Проверить в браузере** — открыть `index.html` через IDE-сервер, убедиться что нет 404, фото отображается, лайтбокс работает.

## Видео в галерее

- Основные — на Vimeo, ссылаются через `videoId` в `photos.json`.
- Локальные превью лежат в `video/` и автоплеятся в галерее (lazy через IntersectionObserver).
- В лайтбоксе видео открывается через iframe Vimeo.

## Цены и контент

- `data/pricing.json` — структура `{ discount, photo: { cards, addon }, video: { cards }, note }`. Скидка применяется автоматически.
- `data/about.json` — текст страницы about.
- Менять контент — только в JSON, HTML не трогать.

## Локальный запуск

Открыть `index.html` через встроенный сервер IDE.

## Деплой

Сайт раздаётся как статика. Достаточно закоммитить и запушить — нет билда.
